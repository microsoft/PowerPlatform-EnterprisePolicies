# EnterprisePolicy

## DESCRIPTION

The `EnterprisePolicy` class represents an Azure ARM Enterprise Policy resource returned by cmdlets like `Get-SubnetInjectionEnterprisePolicy`.

## PROPERTIES

### ResourceId

Type: `System.String`
Description: The full ARM resource ID of the enterprise policy.

### Id

Type: `System.String`
Description: The resource ID (same as ResourceId).

### Kind

Type: `System.String`
Description: The policy type: Encryption, NetworkInjection, or Identity.

### Location

Type: `System.String`
Description: The Azure region where the policy is deployed.

### ResourceName

Type: `System.String`
Description: The name of the resource.

### Name

Type: `System.String`
Description: The name of the policy.

### Properties

Type: `EnterprisePolicyProperties`
Description: The policy-specific properties.

### ResourceGroupName

Type: `System.String`
Description: The resource group containing the policy.

### Type

Type: `System.String`
Description: The ARM resource type (Microsoft.PowerPlatform/enterprisePolicies).

### ResourceType

Type: `System.String`
Description: The ARM resource type.

### SubscriptionId

Type: `System.String`
Description: The Azure subscription ID.

## METHODS

This class does not define any methods.

## EXAMPLES

```powershell
$policy = [EnterprisePolicy]::new()
$policy.ResourceId = "/subscriptions/12345678-1234-1234-1234-123456789012/resourceGroups/myResourceGroup/providers/Microsoft.PowerPlatform/enterprisePolicies/myPolicy"
$policy.Kind = "NetworkInjection"
$policy.Location = "westus"
$policy.Name = "myPolicy"
$policy.ResourceGroupName = "myResourceGroup"
$policy.SubscriptionId = "12345678-1234-1234-1234-123456789012"
```

# EnterprisePolicyProperties

## DESCRIPTION

The `EnterprisePolicyProperties` class contains policy-specific properties including system ID and network injection configuration.

## PROPERTIES

### SystemId

Type: `System.String`
Description: The internal system ID of the policy in the format `/regions/{region}/providers/Microsoft.PowerPlatform/enterprisePolicies/{guid}`.

### NetworkInjection

Type: `NetworkInjectionProperties`
Description: Network injection configuration (for NetworkInjection policies).

## METHODS

This class does not define any methods.

## EXAMPLES

```powershell
$props = [EnterprisePolicyProperties]::new()
$props.SystemId = "/regions/westus/providers/Microsoft.PowerPlatform/enterprisePolicies/12345678-1234-1234-1234-123456789012"
```

# NetworkInjectionProperties

## DESCRIPTION

The `NetworkInjectionProperties` class contains the virtual network configuration for Network Injection Enterprise Policies.

## PROPERTIES

### VirtualNetworks

Type: `VirtualNetworkReference[]`
Description: Array of virtual networks configured for subnet injection.

## METHODS

This class does not define any methods.

## EXAMPLES
