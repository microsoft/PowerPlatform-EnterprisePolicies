# ConnectivityInformation

## DESCRIPTION

The `ConnectivityInformation` class contains the result of a TCP connectivity test performed from your delegated subnet to a destination on a specified port.

## PROPERTIES

### TCPSuccess

Type: `System.Boolean`
Description: Indicates whether the TCP connection to the destination was successfully established.

### TCPErrorMessage

Type: `System.String`
Description: When the connection fails, contains the error message describing why the connection could not be established.

### Destination

Type: `System.String`
Description: The destination hostname or IP address against which the connectivity was tested.

### Port

Type: `System.Int32`
Description: The TCP port against which the connectivity was tested.

### ContainerIpAddress

Type: `System.String`
Description: The IP address of the container in your delegated subnet that performed the connectivity test.

## METHODS

This class does not define any methods.

## EXAMPLES

```powershell
$connectivity = [ConnectivityInformation]::new()
$connectivity.TCPSuccess = $true
$connectivity.Destination = "bing.com"
$connectivity.Port = 443
$connectivity.ContainerIpAddress = "10.1.2.3"
```
