# TLSConnectivityInformation

## DESCRIPTION

The `SSLInformation` class contains detailed information on the TLS handshake attempt.

## PROPERTIES

### Protocols

Type: `System.String`
Description: Contains the protocol used to establish the handshake.

### Success

Type: `System.Boolean`
Description: Indicates whether the handshake was completed successfully.

### SSLErrors

Type: `System.String`
Description: Contains information of errors encountered while trying to establish the handshake.

### CipherSuite

Type: `System.String`
Description: Contains the ciphersuite used to establish the handshake. 

## METHODS

This class does not define any methods.

## EXAMPLES

```powershell
$sslInformation = [SSLInformation]::new()
$sslInformation.Protocols = "TLS1.3"
$sslInformation.Success = $true
$sslInformation.SSLErrors = "None"
$sslInformation.CipherSuite = "TLS_ECDHE_RSA_WITH_AES_128_GCM_SHA256"
```
