# HostResolutionInformation

## DESCRIPTION

The `HostResolutionInformation` class contains the result of a DNS resolution test performed from your delegated subnet against a hostname.

## PROPERTIES

### Success

Type: `System.Boolean`
Description: Indicates whether the hostname was resolved successfully.

### HostName

Type: `System.String`
Description: The hostname that was resolved.

### IPAddresses

Type: `System.String[]`
Description: When resolution succeeds, the IP addresses returned for the hostname.

### DnsServers

Type: `System.String[]`
Description: The DNS servers that were used to attempt the resolution.

### ErrorMessage

Type: `System.String`
Description: When resolution fails, contains the error message describing why the hostname could not be resolved.

### ContainerIpAddress

Type: `System.String`
Description: The IP address of the container in your delegated subnet that performed the resolution.

## METHODS

This class does not define any methods.

## EXAMPLES

```powershell
$resolution = [HostResolutionInformation]::new()
$resolution.Success = $true
$resolution.HostName = "bing.com"
$resolution.IPAddresses = @("13.107.21.200")
$resolution.DnsServers = @("10.0.0.10")
$resolution.ContainerIpAddress = "10.1.2.3"
```
