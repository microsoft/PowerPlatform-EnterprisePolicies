# AzureEnvironment

## DESCRIPTION

Represents the different Azure environments that can be used to connect to Azure services. Only environments that are currently supported are included.

## PROPERTIES

| Enum Value Name | Description |
|-----------------|-------------|
| AzureCloud | Azure Public cloud environment, cmdlets will default to this value. |
| AzureChinaCloud | Azure China cloud environment. |
| AzureUSGovernment | Azure Government cloud environment (includes GCC High and DoD). |
| EastUs2Euap | Azure Early Update Access Program (EUAP) environment in East US 2 region. Microsoft internal use only. |
| CentralUSEuap | Azure Early Update Access Program (EUAP) environment in Central US region. Microsoft internal use only. |

### EXAMPLE 1

$environment = [AzureEnvironment]::AzureCloud
