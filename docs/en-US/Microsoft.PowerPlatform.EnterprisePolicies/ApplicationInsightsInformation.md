# ApplicationInsightsInformation

## DESCRIPTION

The `ApplicationInsightsInformation` class contains the result of an Application Insights connection test performed from your delegated subnet. The test validates the supplied connection string and attempts to send a test telemetry event to the configured Application Insights resource.

## PROPERTIES

### ConnectionStringValid

Type: `System.Boolean`
Description: Indicates whether the supplied Application Insights connection string was successfully parsed and validated.

### TestMessageSent

Type: `System.Boolean`
Description: Indicates whether the test telemetry event was successfully delivered to the Application Insights ingestion endpoint.

### ErrorMessage

Type: `System.String`
Description: When the connection string is invalid or the test event fails to send, contains the error message describing the failure.

### ContainerIpAddress

Type: `System.String`
Description: The IP address of the container in your delegated subnet that performed the test.

## METHODS

This class does not define any methods.

## EXAMPLES

```powershell
$information = [ApplicationInsightsInformation]::new()
$information.ConnectionStringValid = $true
$information.TestMessageSent = $true
$information.ContainerIpAddress = "10.1.2.3"
```
